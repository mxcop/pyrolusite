# Pyrolusite
My personal static blog generater written in Rust.
[<sup>Demo</sup>](https://mxcop.github.io/pyrolusite)

<br>

## Post format
```
{{ title }}
{{ day/month/year }}

{{ description }}

---

{{ markdown }}
```

<br>

## Build
This command will build all markdown files inside `./md/` into `./dist/`.
```sh
$ cargo run
```
